Логические операции

Эта программа тестирует, находится ли переменная a в диапазоне от пяти до десяти. Используется логический оператор and (логическое И): 

program hello;

var a: integer;

begin
    a := 8;
    writeln ((a >= 5) and (a <= 10));
    
    readln
end.

Выражение (a >= 5) and (a <= 10) можно разбить на части. В левой скобке a сравнивается с пятью. >=, как и любая операция сравнения — это операция, которая на вход принимает два числа, а на выход возвращает логическое значение TRUE (правда) или FALSE (ложь). Аналогичная операция сравнения (меньше-или-равно) находится в правой скобке. Посередине расположена операция логического И: она возвращает TRUE тогда и только тогда, когда и левая, и правая часть операции равны TRUE, иначе возвращается FALSE.

Циклы

Цикл repeat-until выполняется, пока условие в until не станет ложным. 

    repeat { повтори }
        { шаг 1;     }
        { шаг 2;     }
        { ...       }
    until COND { пока условие COND не станет ложным}

Цикл for-do, переменную i надо объявить заранее: 

    for i := 1 to 10 do { для переменной i от одного до десяти повтори }
    begin   { начало цикла }
        { шаг 1;    }
        { шаг 2;    }
        { ...       }
    end     { конец цикла }

Ежели в for-do цикле подразумевается только один шаг, то begin и end можно опустить: 

    for i := 10 downto 1 do { для переменной i от десяти до одного повтори }
    begin
        { единственный шаг цикла;    }

Два разных цикла, которые считают сумму от одного до 60: 

program series;

var sum: integer;
    counter: integer;
    i: integer;

begin
    sum := 0;
    counter := 0;
    repeat
        counter := counter + 1;
        sum := sum + counter;
    until counter >= 60;
    writeln(sum);
    sum := 0;
    for i := 1 to 60 do
        sum := sum + i;
    writeln(sum);
    readln
end.

Графика

Первая графическая программа, рисует окружность в левом верхнем углу: 

program gfirst;

uses graph;

var device, mode: integer;

begin
    device := 0;
    initgraph(device, mode, 'C:\TP\BGI');
    circle(100, 80, 50);
    readln;
    closegraph
end.

Первая анимация: используется цикл и условный оператор: 

program animation;

uses crt, graph;

var device, mode: integer;
    x, y: integer;
    dx, dy: integer;

begin
    detectgraph(device, mode);
    initgraph(device, mode, 'C:\TP\BGI');
    x := 0;    y := 0; dx := 1; dy := 1;
    setcolor(green);
    repeat
        circle(x, y, 15);
        x := x + dx;
        y := y + dy;
        delay(10);
        if (y > 300) or (y < 0) then
           dy := - dy;
        if (x > 400) or (x < 0) then
           dx := -dx;
    until keypressed;
    closegraph;
end.
