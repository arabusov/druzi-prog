Типы данных

Внутри компьютера электрические сигналы соответствуют числам в соответствии с определённой договорённостью. Может быть такая договорённость: если напряжение на одном проводке от нуля до двух Вольт, то это нуль, если от трёх до пяти — то это один. Тогда несоклько проводков можно использовать для кодирования двоичных чисел. В общем случае правило, по которому набору сигналов придают какое-то особое значение, называют типом данных.

В Паскале есть несколько типов для целых числе в зависимости от их размера, тип для действительных чисел, тип для логических выражений, и тип для символов.

Для Турбо Паскаля используйте следующие типы переменных: byte для целых чисел в диапазоне 0–255, word в диапазоне 0–65535, integer в диапазоне -32768–32767, boolean для логических значений TRUE или FALSE, real для действительных чисел и char для символов. 

program test_integer;

var x: integer;     { попробуйте поменять integer на другой тип и посмотрите
                      что произойдёт. }

begin
    x := 2;
    repeat
        writeln(x);
        x := x*2;
    until x = 0;
    readln;
end.

Клавиатурный ввод

program keylogger;

uses crt;

var
    ch: char;                       { тип char используется для символов}

begin
    repeat
        if keypressed then          { если какая-то клавиша нажата...    }
        begin
            ch := readkey;          { readkey читает символ клавиши }
            write(ord(ch), ' ');    { ord выдаёт числовой код для символаa }
            if ch = #0 then { некоторые клавиши генерируют два символа на нажатие }
            begin           { первый из них будет всегда нулём      }
                ch := readkey;
                write(ord(ch));
            end;
            writeln;
        end
    until ch = #27      { #27 означает клавишу ESC }
end.
