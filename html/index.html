  • Termux — это эмулятор командной строки для телефонов на базе Android без необходимости дополнительных настроек или доступа к суперпользователю на телефоне. На новых Android не работает.

  • PuTTY — это программа для Windows, которая позволяет подключаться через сеть к серверам на Linux. Последнюю версию можно скачать по этой ссылке.

  • QEMU — это программа, которая эмулирует компьютер вниутри компьютера. QEMU может эмулировать разные типы компьютеров: так, например, если у вас нет компьютера с Linux, но хочется поэкспериментировать и есть компьютер с Windows, то QEMU подходит как нельзя кстати для таких опытов. Скачать QEMU для Windows можно по ссылке.

    Готовый образ Slackware Linux 15.0 для QEMU можно скачать по ссылке (внимание, размер файла 15 гигабайт!) Для запуска на своей системе я использую следующую команду для запуска образа и эмуляции сети внутри виртуального компьютера: 

    qemu-system-x86_64 \
    -hdd sl15.img\
    -accel kvm \
    -netdev user,id=n1 -device virtio-net-pci,netdev=n1 \
    -m 4G
    Однако в Windows команда должна выглядеть по-другому.
