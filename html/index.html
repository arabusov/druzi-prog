Ресурсы:

  • DOSBox — это эмулятор простой среды MS-DOS. Идеально подходит для изучения простых приёмов программирования и графики.

  • DOSBox for Android — это DOSBox для Android. Можно изучать программирование прямо в телефоне!

  • GitHub — это хостер проектов с открытым исходным кодом. Ссылка ведёт на исходный код этого курса.

  • PC JS — это эмулятор IBM PC в браузере. Ссылка ведёт на образ с Borland Turbo Pascal 6.00.

  • Termux — это эмулятор командной строки для телефонов на базе Android без необходимости дополнительных настроек или доступа к суперпользователю на телефоне. На новых Android не работает.

  • PuTTY — это программа для Windows, которая позволяет подключаться через сеть к серверам на Linux. Последнюю версию можно скачать по этой ссылке.

  • JSLinux — это эмулятор Linux прямо в браузере, отличный старт для тех кто не хочет возиться с установкой эмулятора на свой компьютер.

  • QEMU — это программа, которая эмулирует компьютер вниутри компьютера. QEMU может эмулировать разные типы компьютеров: так, например, если у вас нет компьютера с Linux, но хочется поэкспериментировать и есть компьютер с Windows, то QEMU подходит как нельзя кстати для таких опытов. Скачать QEMU для Windows можно по ссылке. Для самостоятельных занятий вы можете использовать мой образ диска (описание здесь).
