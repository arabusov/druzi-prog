Установка и настройка QEMU

Хорошее описание того как настроить QEMU вы можете найти здесь. Следуйте инструкциям до настроек переменных среды (environment variables) включительно. Для того чтобы протестировать работу QEMU, откройте Windows Power Shell (from the Start menu, click Start, click All Programs, click Accessories, click the Windows PowerShell folder, and then click Windows PowerShell), и в нём выполните команду 

qemu-system-x86_64

Должно появиться окно с виртуальной машиной. QEMU покажет примерно вот такие вот сообщения на экране (версия BIOS будет скорее всего другой):

Запуск Slackware Linux 15.0 в QEMU

Готовый образ Slackware Linux 15.0 для QEMU можно скачать по ссылке (внимание, размер файла 15 гигабайт!) Важно запомнить, в какую папку вы скачали этот файл. Для запуска следующую команду для запуска образа и эмуляции сети внутри виртуального компьютера: 

qemu-system-x86_64 -hdd ПУТЬ_ДО_ОБРАЗА\sl15.img -m 4G

На экране сначала появится такая надпись: 

SeaBIOS (version Arch Linux 1.16.2-1-1)


iPXE (http://ipxe.org) 00:03.0 C900 PCI2.10 PnP PMM+BEFD3390+BEF33390 C900



Booting from Hard Disk...

LILO 24.2

И через пять секунд начнётся загрузка Slackware Linux. Если загрузка пройдёт успешно, Slackware Linux будет приветствовать следующим сообщением, запрашивая вас ввести имя пользователя и пароль: 

Welcome to Linux 5.15.94 x86_64 (ttyS0)

sl15 login:

В образе sl15.img доступен только суперпользовател root с паролем toor. Сначала наберите имя пользователя, нажмите Enter, затем введите пароль (и тоже нажмите Enter). При вводе пароля никаких символов выводиться на экран не будет. При успешном входе в систему вы увидите системное приглашение: 

root@sl15:~#

Теперь вы можете следовать документации для новичков: slackdoc. Например, вам следует создать нового пользователя, разлогиниться из-под root и залогиниться снова с новым пользователем.

Для выключения виртуальной машины надо залогиниться под root и ввести команду halt.
