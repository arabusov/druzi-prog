Запуск QEMU и Slackware Linux 15.0

Готовый образ Slackware Linux 15.0 для QEMU можно скачать по ссылке (внимание, размер файла 15 гигабайт!) Для запуска на своей системе я использую следующую команду для запуска образа и эмуляции сети внутри виртуального компьютера: 

qemu-system-x86_64 \
    -hdd sl15.img\
    -accel kvm \
    -netdev user,id=n1 -device virtio-net-pci,netdev=n1 \
    -m 4G
Однако в Windows команда должна выглядеть по-другому.

После успешного запуска Slackware Linux 15.0 пригласит вас ввести имя пользователя и пароль: 

Welcome to Linux 5.15.94 x86_64 (tty1)

sl15 login:

В образе sl15.img доступен только суперпользовател root с паролем toor. Сначала наберите имя пользователя, нажмите Enter, затем введите пароль (и тоже нажмите Enter). При вводе пароля никаких символов выводиться на экран не будет. При успешном входе в систему вы увидите системное приглашение: 

root@sl15:~#

Теперь вы можете следовать документации для новичков: slackdoc, например, вам следует создать нового пользователя, разлогиниться из-под root и залогиниться снова с новым пользователем.

Для выключения виртуальной машины надо залогиниться под root и ввести команду halt.
